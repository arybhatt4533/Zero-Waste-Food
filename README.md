# Zero Waste Food 🌿

Zero Waste Food is a web-based platform dedicated to rescuing excess food and distributing it to those in need, aiming to reduce food waste across India.

## 🚀 How to Setup and Run Locally

Follow these steps to set up the project on your machine:

### 1. Prerequisites
Ensure you have the following installed:
* [Node.js](https://nodejs.org/) (v16+)
* [Git](https://git-scm.com/)
* [PostgreesSQL](https://www.mongodb.com/) (Local or Atlas)

### 2. Clone the Repository
```bash
git clone [https://github.com/arybhatt4533/Zero-Waste-Food.git](https://github.com/arybhatt4533/Zero-Waste-Food.git)
cd Zero-Waste-Food

### 3. Install Dependencies
You need to install packages for both the client and server.
cd server
npm install
### 4.  For client
cd ../client
npm install
### 5. Running the Application
Start the Server: node server.js
Open a terminal in the server folder:
### 6. Open new Terminal
Start the Client: npm start
Open a new terminal in the client folder:

Tech Stack
Frontend: React.js

Backend: Node.js, Express.js
